"HTML, diferente do que a maioria das pessoas pensa, não é uma linguagem de programação mas sim uma estrutura de marcação de textos. Na prática o HTML é responsável pela estruturação da página, o esqueleto do que será visto no seu navegador"

HTML
Linguagem de marcação de texto

"Aqui sim começamos a falar mais sério sobre programação. A linguagem Javascript é a responsável por contruir a lógica por trás de uma página, essa linguagem permite, entre outras coisas, determinar que imagens de elemntos de uma página (thumbs de notícias por exemplo) sejam carregados somente quando a barra de rolagem chega até esses elementos, agilizando o carregamento da página e sua navegaçnao. Mas issó é apensas uma amostra do que o Javascript é capaz."
